#### 数组

定义： **① int[] a = new int[5];**

​			② int a[] = new int[5];

​			③ int a[] = {1,2,3,5,2}; 静态

数组长度：array.length

> 数组兼容类型。

数组中默认值

- int、short、byte、long全为 0
- float、double 为0.0
- char 为 \u0000 （空格）
- boolean 为 false
- String 为 null
- 对象为null



##### 数组赋值注意：(地址传递)

```java
int[] arr1 = {1,2,3};
int[] arr2 = arr1;
arr2[0] = 10;
//输出的值arr1[0]=arr2[0]=10
```



##### 二维数组 

```java
int [][] arr = {{1,1},{2,2}};
```

其他的定义方式：(把int[]看作一种数据类型，即可理解)

```java
int[] arr[] = {{1,1},{2,2}};
int arr[][] = {{1,1},{2,2}};
```

**可以先只规定行数，列数不确定。**

**即：二维数组它的各个一维数组的长度可以不相同。**

**（列数后面自行，每一行定义开辟自己想要的长度）**

如： 1
